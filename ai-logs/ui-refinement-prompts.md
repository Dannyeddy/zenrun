# UI Refinement Prompts

## Mobile-First UI

Refine the ZenRun interface for a mobile-first coursework demo. Keep the rounded white-card style, improve visual hierarchy, and make sure common actions are easy to tap.

## Atlas Map Adjustments

Fix the World Atlas and China region postcard positioning. Keep the world view unchanged, but adjust China region `xRegion` and `yRegion` coordinates so the postcards sit in the upper-left area of the zoomed map and are visually spread out.

## Companion Card Popup

Update the Dashboard Companion card so it opens a cute daily companion message popup instead of duplicating the Profile page. The message should use the pet's first-person voice and connect weekly plan progress with food, energy, and treasure fragments.

## Reward Page Visual Polish

Improve the post-run reward result pages so the user clearly understands what they earned, why it matters, and how it connects to the pet or Atlas collection.

## Dashboard Polish

Polish the Home dashboard so it clearly shows weekly progress, companion status, recommended routes, and a strong tap-to-start route card without redesigning the whole app.

## Tracker Overlay Cleanup

Remove the repeated "Discovery Triggered / Historical Route Ready" visual popup from the Tracker while preserving run progress, memory pieces, mood music, voice narration, and completion logic.

## China-Accessible Tracker Map

Update the Tracker map visual tile source so it is more reliable in mainland China. Keep route progress and landmark distance calculations separate from the visual map layer and provide a fallback if live tiles fail.
