# AI Development Logs

This folder contains representative prompts used during AI-assisted development of ZenRun.

AI tools were used for:

- Code scaffolding and feature planning
- UI refinement and mobile layout polish
- Debugging build and interaction issues
- Reward, fragment, and Atlas map logic support
- Dashboard data binding, check-in calendar logic, and companion reminder logic
- Mood music playback lifecycle and map-source reliability checks
- Route-specific share cards and public share link generation
- Cultural relic treasure naming and Collection deduplication
- Data handling documentation
- Coursework submission preparation

The team reviewed, tested, and manually adjusted AI-generated outputs before including them in the final prototype. These logs are representative rather than exhaustive. They do not include private tokens, API keys, passwords, or sensitive account information.
